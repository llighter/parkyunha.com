VS Code에서 파일을 생성할 때 기본적으로 제공하는 단축키는 `⌘ + N`이다. 이 단축키를 사용하면 새 파일이 생성되지만,
저장할 때 다시 파일 위치와 이름을 입력해야 한다. 이 과정이 번거롭다면 단축키를 설정하여 바로 파일을 생성하고 이름을 지정하는 방법을 알아보자.

순서는 아래와 같고 위 캡쳐본을 참고하면 이해하기 쉽다.

1. **단축키 설정 열기**: `⌘ + K`를 누른 후 `⌘ + S`를 눌러 단축키 설정을 연다.
2. **명령어 검색**: 검색창에 `explorer.new` 을 입력하면 두 가지 항목이 나타난다.
3. **단축키 설정(파일)**: `Explorer: New File` 항목을 찾아 원하는 단축키를 설정한다. 예를 들어 `⌘ + N`을 설정할 수 있다.
4. **단축키 설정(폴더)**: `Explorer: New Folder` 항목을 찾아 원하는 단축키를 설정한다. 예를 들어 `⇧ + ⌘ + N`을 설정할 수 있다.
5. **저장**: 설정을 완료한 후, 변경 사항을 저장한다.
6. **파일 생성**: `⇧ + ⌘ + E`를 눌러 탐색기를 열고, 설정한 단축키로 파일이나 폴더를 생성한다.
